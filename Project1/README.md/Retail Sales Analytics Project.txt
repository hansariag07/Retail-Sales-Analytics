# Retail Sales Analytics Project

## Project Overview
This project is an end-to-end retail sales analytics solution built to analyze customer and sales data using industry-standard tools.

## Tools & Technologies
- Excel – Raw data source
- MySQL – Database, joins, and analysis
- Python (Pandas, Matplotlib) – Data preprocessing and analysis
- Power BI – Interactive dashboard and visualization
- Git & GitHub – Version control and project hosting

## Project Workflow
1. Collected customer and order data in Excel
2. Imported data into MySQL with primary and foreign key relationships
3. Performed SQL joins and business queries
4. Extracted data into Python for preprocessing and EDA
5. Built an interactive Power BI dashboard for insights
6. Uploaded the complete project to GitHub

## Key Analysis Performed
- Total Revenue, Orders, Customers, and Average Order Value
- City and State-wise sales performance
- Category and Product-wise revenue analysis
- Monthly sales trend analysis
- Customer-level sales insights

## Output
- Cleaned dataset ready for reporting
- Interactive Power BI dashboard
- Structured and documented GitHub project
