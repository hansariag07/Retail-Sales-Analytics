# Retail Sales Analytics Project

## Overview
This project demonstrates an end-to-end retail sales analytics workflow using Excel, SQL, Python, and Power BI to derive business insights from customer and sales data.

## Tools Used
- Excel – Raw data storage
- MySQL – Data modeling, joins, and analysis
- Python (Pandas, Matplotlib) – Data preprocessing and exploratory analysis
- Power BI – Interactive dashboard and visualization
- Git & GitHub – Version control and project hosting

## Project Workflow
1. Customer and order data collected in Excel
2. Data imported into MySQL with primary and foreign key relationships
3. SQL joins used to create an analytical dataset
4. Data extracted into Python for cleaning and analysis
5. KPIs and trends visualized using Power BI
6. Project published on GitHub

## Key Analysis
- Total Revenue, Orders, Customers, and Average Order Value
- City and State-wise sales performance
- Category and Product-level revenue analysis
- Monthly sales trend analysis

## Output
- Cleaned dataset for reporting
- Interactive Power BI dashboard
- Structured analytics project on GitHub
